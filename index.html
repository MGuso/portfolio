Automatic inspection system for metal parts using artificial intelligence

This project involved designing an inspection system for metal parts with machine vision for a machining company. A set of operations is applied to the parts simultaneously through a die machining process. The issue arises when some operations fail due to the spontaneous breakage of the die, and the resulting defect goes unnoticed. The proposed solution involves processing images of the parts using an artificial neural network. A camera captures an image of the part, which is then processed to determine if the areas where an operation should have been performed are actually completed. The neural network has been trained using deep learning technologies and a set of example part images. The inspection results are displayed to the operator on a screen.

Keywords: Artificial Intelligence, Convolutional Neural Networks, Deep Learning, Machine Vision, Image Classification, Automation, Python, OpenCV.

Carbohydrate estimation algorithm from food images using artificial intelligence

The system is able to estimate the amount of carbohydrates from a photograph of a dish using a regression model based on a transformer-type neural network. In this project, an algorithm is programmed to detect the amount of carbohydrates present in a dish from its photograph. The solution is based on using a transformer neural network to process images and create a regression model. The goal of the system is to assist Type 1 diabetes patients, who need to monitor various factors that affect the amount of insulin they should administer at any given time.

Keywords: Artificial Intelligence, Deep Learning, Transformer Neural Networks, Machine Vision, Regression, Python.

Defect detection model in industrial biscuit production

This project aims to apply quality control in industrial cookie production by using product images to detect potential defects. The solution is based on a convolutional neural network trained with deep learning technologies. The training is carried out on a sample dataset, which includes images of both correct and defective samples. Three different types of defects are identified.

Keywords: Artificial Intelligence, Deep Learning, Convolutional Neural Networks, Machine Vision, Automation, Python, OpenCV.

Word-matching game based on meaning using artificial intelligence

A game programmed in Python that uses OpenAI's GPT-2 artificial intelligence model, based on NLP (Natural Language Processing) technology. The model sorts a large set of words based on their semantic proximity to a randomly chosen first word. The game involves discovering the first word in the hidden list by blindly guessing words and finding their position on the list. The objective of the game is to identify and input the first word from the hidden list.

Keywords: Artificial Intelligence, Deep Learning, Natural Language Processing, Word Embedding, Python, NumPy.
